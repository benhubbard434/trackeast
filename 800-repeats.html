800m repeats

Most running tracks are 400m in length, so 800m repeats is a pretty simple session. It's two laps of the track for each rep. 800m is roughly equivalent to half a mile.

Before we start the workout, someone will lead the group in some drills. Drills are short exercises that help us to improve our running form, and commit running movement patterns to muscle memory.

Before the drills, we will all run a few warm-up laps together (aim to warm up for a minimum of ten minutes). The warm-up should always be at a pace that feels really easy: you should be able to hold a conversation with the person running next to you without being too out of breath.

If you're new to track, build gradually. Perhaps start with slightly fewer repeats or a slightly longer rest between each rep. Also, feel free to approach anyone else if you have questions, we're all happy to help.

THE WORKOUT:

Try to complete as many 800m reps as you can fit into 40 minutes, with a 90-second static recovery between each rep.

Static recovery simply means that between each lap you stand still (rather than active recovery, which would be a walk or very easy jog while your heart rate comes down).
